## プライバシーポリシー

Bottlemintは、静かで匿名な体験を大切にしています。
本サービスでは、必要最小限の情報のみを取り扱います。

### 収集する情報
- アカウント登録は不要です。
- 氏名、メールアドレス、プロフィール情報は収集しません。
- メッセージは匿名で保存されます。

### 技術的情報について
サービス運営および不正利用防止のため、以下の情報を一時的に取り扱う場合があります。
- 生成されたデバイス識別子
- アクセスログ（日時、リクエスト情報など）

これらは、サービスの維持・改善および安全確保の目的にのみ使用されます。

### 利用目的
- メッセージの配信・表示
- サービスの運営および改善
- スパムや不正利用の防止

広告目的やトラッキングには使用しません。

### 第三者提供
- 個人情報を販売・共有することはありません。
- 法令に基づく場合を除き、第三者に提供することはありません。

### データの保持
- データは、サービス運営に必要な期間のみ保持されます。
- メンテナンス等により、予告なく削除される場合があります。
